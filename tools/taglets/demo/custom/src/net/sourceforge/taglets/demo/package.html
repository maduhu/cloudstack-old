Demo sources for the Taglets Collection.

Demo Sources

The sources in this package show off some of the default tags supported by the Taglets Collection.

There are two major categories of tag types supported:

net.sourceforge.taglets.demo.inline
Inline tags, like {@link net.sourceforge.taglets.demo.inline.Stickies#stickyInfo {@stickyInfo}}.
net.sourceforge.taglets.demo.block
Block tags, like @adm.
